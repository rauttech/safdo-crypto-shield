# safdo-crypto-shield
Safdo Crypto Shield is a next-generation Web3 security toolkit for the Polkadot ecosystem. It allows users to connect their Polkadot.js wallet, analyze multi-chain activity, upload smart contracts to IPFS, and generate security insights — all built using Flutter Web + Polkadot RPC.
